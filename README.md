# frontbox
Static Site Generators

![frontbox](http://i.imgur.com/vZTp0QW.png)

# Install
`$ npm install frontbox -g`

# Usage
Choose one with them

    $ frontbox [stylesheet]

# stylesheet

1. sass
2. scss
3. styl

install npm package

    $ npm install

after that you must run

    $ frontbox fixmd

and run

    $ gulp
